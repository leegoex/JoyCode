ImageMagick学习笔记

使用ImageMagick主要是为了用来在Linux进行图片的处理。我需要对图片进行以下处理：

  1. 截取图片上指定区域生成新的图片
  2. 缩放图片
  3. 在图片上叠加文字
  4. 合并多张图片

一、截取图片生指定区域生成新的图片.使用Image类的crop可以截取图片指定区域的部分。该函数的原型为：
void Magick::Image::crop(const Geometry& geometry_)
该函数接收一个类型为const Geometry& 的变量作为参数，用来指定需要截取的宽、高和起始坐标。

二、缩放图片。使用Image类的zoom可以对一个图片进行缩放。该函数的原型为：
void Magick::Image::zoom(const Geometry& geomerty_)
该函数接收一个类型为const Germetry& 的变量作为参数，用来指定缩放后图片的宽、高。

三、在图片上叠加文字。使用Image类的draw可以在图片上绘制任何可以绘制的内容。该函数的原型为：
void Magick::Image::draw(const std::list<Magick::Drawable>& drawable_)
drawable_是一个可绘制对象的链表。Magick++定义了许多可绘制的对象，例如DrawableColor、DrawableFont、DrawableText等。

四、合并多张图片。使用模板函数appendImages可以将多个图片合并成一张图片。该函数的原型为：
template< class InputIterator >
void Magick::appendImages(Image* appendedImage_, InputIterator first_, InputIterator last_, bool stack_=false)
appendedImage_返回合并后的图片
first_、last_为要合并图片的迭代起始位置和结束位置。所以，首先要将要合并的图片放在一个可迭代的对象中，例如链表。
stack_为合并方式，false表示水平合并，true表示垂直合并。默认为水平合并
